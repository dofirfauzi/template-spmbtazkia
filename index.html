Alur Pendaftaran
Test Potensi Akademik

  • 1. Daftar Buat Akun

    Untuk melakukan pendaftaran, Calon pendaftar harus membuat akun di web spmb.tazkia.ac.id

  • 2. Isi Formulir Pendaftaran

    Login di web spmb.tazkia.ac.id/login kemudian isi formulir pendaftaran

  • 3. Lakukan Pembayaran Biaya Pendaftaran

    Tagihan biaya pendaftaran dapat di cek di dashboard pendaftar pada menu tagihan. Atau di Email yang sudah terdaftar

  • 4. Isi Data Detail

    Login di web spmb.tazkia.ac.id/login kemudian isi data detail anda pada menu data detail

  • 5. Pilih Jalur & Jadwal Test

    Pada menu jalur test pilih TPA (Test Potensi Akademik) lalu pilih hari tanggal dan waktu test yang diinginkan.

  • 6. Ikuti Test

    Ikuti test sesuai jadwal yang telah di pilih, test dapat dilaksanakan di kampus tazkia atau online.

  • 7. Pengumuman Hasil Test

    Hasil test dapat dilihat di dashboard pendaftar pada menu hasil test

  • 8. Daftar Ulang

    Lakukan pembayaran biaya daftar ulang. Untuk mendapatkan tagihan daftar ulang silahkan hubungi admin.

Alur Pendaftaran
Jalur Prestasi Akademik

  • 1. Daftar Buat Akun

    Untuk melakukan pendaftaran, Calon pendaftar harus membuat akun di web spmb.tazkia.ac.id

  • 2. Isi Formulir Pendaftaran

    Login di web spmb.tazkia.ac.id/login kemudian isi formulir pendaftaran

  • 3. Lakukan Pembayaran Biaya Pendaftaran

    Tagihan biaya pendaftaran dapat di cek di dashboard pendaftar pada menu tagihan. Atau di Email yang sudah terdaftar

  • 4. Isi Data Detail

    Login di web spmb.tazkia.ac.id/login kemudian isi data detail anda pada menu data detail

  • 5. Pilih Jalur & Jadwal Test

    Pada menu jalur test pilih JPA (Jalur Potensi Akademik) lalu pilih tanggal test (pengiriman berkas) yang diinginkan.

  • 6. Kirim Berkas

    Kirim berkas JPA (Raport semester 1 - 4, Sertifikat Penghargaan & Surat Rekomendasi dari Sekolah) via email ke spmb@tazkia.ac.id

  • 7. Pengumuman Hasil Test

    Hasil test dapat dilihat di dashboard pendaftar pada menu hasil test paling lambat 1 minggu setelah berkas di kirim.

  • 8. Daftar Ulang

    Lakukan pembayaran biaya daftar ulang. Untuk mendapatkan tagihan daftar ulang silahkan hubungi admin.

Alur Pendaftaran
Smart Test

  • 1. Mengikuti SmartTest di Sekolah

    Pastikan anda telah mengikuti test di sekolah yang telah bekerjasama dengan tazkia dan dinyatakan lulus (lihat hasil test disini)

  • 2. Daftar dan Buat Akun

    Isi form pendaftaran Smart Test dengan cara klik tombol daftar pada kolom action di tabel hasil SmartTest

  • 3. Lakukan Pembayaran Biaya Pendaftaran

    Tagihan biaya pendaftaran dapat di cek di dashboard pendaftar pada menu tagihan. Atau di Email yang sudah terdaftar

  • 4. Isi Data Detail

    Login di web spmb.tazkia.ac.id/login kemudian isi data detail anda pada menu data detail

  • 5. Daftar Ulang

    Lakukan pembayaran biaya daftar ulang. Untuk mendapatkan tagihan daftar ulang silahkan hubungi admin.

Alur Pendaftaran
Jalur Beasiswa

  • 1. Daftar Beasiswa

    Lakukan pendaftaran seleksi beasiswa melalui web beasiswa.tazkia.ac.id. untuk info beasiswa hubungi (kak barly) +62 821-1528-6212

  • 2. Ikuti Seleksi Beasiswa

    Setelah mendaftar silahkan ikuti proses seleksi beasiswa hingga akhir

  • 3. Daftar Ulang

    Jika dinyatakan lulus seleksi beasiswa hubungi admin penerimaan mahasiswa baru untuk melakukan proses daftar ulang dengan memberikan surat keterangan hasil seleksi beasiswa.
